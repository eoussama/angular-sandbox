# Angular Sandbox

Sample Superhero project for the sake of teaching. It uses all the mundane Angular features that you might run across. (Count stopped at ng15).

## Usage

1. Clone the project
    ```sh
    git clone git@github.com:eoussama/angular-sandbox.git
    ```

2. Install all dependencies
    ```sh
    npm i
    ```

3. Run the API
    ```sh
    npm run api
    ```

4. Run the app
    ```sh
    npm start
    ```


### Notice
> Default credentials are `admin/123`.

## Testing

### Coverage

```sh
npm run cov
```

### Unit Tests

```sh
npm test
```

### E2E Tests

```sh
npm run e2e
```

### Storybook

```sh
npm run storybook
```
